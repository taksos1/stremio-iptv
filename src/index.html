About the Addon

This addon integrates IPTV playlists and TV guides into Stremio. Your configuration is encoded into a token so you can reinstall or share securely.

  • M3U + XMLTV (Direct)
  • Xtream Codes JSON / m3u_plus
  • Optional custom EPG + offset
  • Channel, Movie & Series catalogs
  • Local + Redis caching support
